# TodoList  
To implement a todolist website using React &amp; Django REST Framework  

---

#### Link to My Website:  
[http://balibaba.top:8000/todo](http://balibaba.top:8000/todo)

---
<br>

#### 功能实现：

* [x] 增加一个待办事项  
* [x] 删除一个待办事项  
* [x] 标记一个待办事项为已完成  
* [x] 编辑一个待办事项的具体内容  
* [x] 列出所有的待办事项  
* [x] 项目已部署到阿里云
<br>

#### 技术实现：
后端：Django  
框架：Django Rest Framework  
前端：React + Bootstrap + AntD  
数据库：PostgreSQL  

~~(已使用Webpack打包)~~
